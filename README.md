📖 Descrição

Este projeto reúne 10 exercícios práticos em C e Python, com foco em conceitos fundamentais de programação, como:

Controle de fluxo
Recursividade
Escopo e memória
Passagem de parâmetros
Mutabilidade e efeitos colaterais
🎯 Objetivo

Analisar como diferentes abordagens de programação impactam o desempenho, a organização do código e o controle de estado.
